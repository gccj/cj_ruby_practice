##### 1から入力した数値までの合計を計算しましょう
````
ruby goukei.rb 10
>> 55

````

##### パスワードジェネレータ
````
ruby passwd.rb 10
>> WXtvAfFajV

````

##### 受け取ったファイルの行数を数えて表示する
````
ruby line_count.rb cal.rb 
>> 70

````

##### 最初の文字と最後の文字が同じ文字列をカウント
````
ruby match_ends.rb aba xyz aa x bbb
>> 3

````

##### 配列をソート
````
ruby sort.rb [1,2,4,10,30,20,3,5,6] (1,2,4,10,30,20,3,5,6 or 1 2 4 10 30 20 3 5 6)
>> [1, 2, 3, 4, 5, 6, 10, 20, 30]
````
##### 配列の最後の値でソートしよう
````
# array: [[1, 7], [1, 3], [3, 4, 5], [2, 2]]
ruby sort_last.rb
>> [[2, 2], [1, 3], [3, 4, 5], [1, 7]]

````

##### 入力したものの整数値判断
````
# 整数の場合
ruby is_int.rb 15
>> 整数値が入力されました

# 整数以外の場合
ruby is_int.rb 0.3
>> 整数値以外が入力されました

````

##### 入力したものの小数値判断
````
# 小数の場合
ruby is_float.rb .3
>> 小数値が入力されました


# 小数以外の場合
ruby is_float.rb 15
>> 小数値以外が入力されました

````
